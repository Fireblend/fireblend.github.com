Hello! This is an overview of how The Quiet Bot works. The Quiet Bot is a Discord helper bot that facilitates common player tasks in the Tabletop Role-Playing Game "The Quiet Year" by Avery Alder and others with mechanics similar to it such as "Anomaly" by Carter Richmond.
It contains deck (52-card and tarot) shuffling and dealing, project management and general purpose commands such as note-taking and dice rolling.
And it's made by me, Sergio Morales (www.fireblend.com)

The Quiet Bot BOT 02-Apr-20 07:36 AM
=== Basic Commands === !start: Reshuffles the decks and clears the project, contempt and notes list. !draw: Draws a card from the standard 52-card deck. !tarot: Draws a card from the tarot deck. === Project Commands === !project <num> <description>: Starts a <num>-step project with the <description> given. !projects: Lists all active projects. !projects all: Lists all projects, including cancelled and finished ones. !advance: Advances all projects by one step. !cancel <id>: Cancels project with the given <id>. !finish <id>: Marks the project with the given <id> as finished. !set <id> <steps>: Sets the project with the given <id> to <steps> steps. Only works on active projects. === Other Commands === !roll <num>d<sides>: Rolls <num> dice of <sides> sides. !note <text>: Stores the <text> into the notes. !notes: Shows all notes. !contempt: Adds a contempt token to your player counter. !contempts: Shows contempt counters.
